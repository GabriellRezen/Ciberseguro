## 🇵🇹 CiberSeguro

**CiberSeguro** é uma plataforma web desenvolvida em 72 horas no âmbito de um hackathon proposto pela **PSP (Polícia de Segurança Pública)**, com o objetivo de combater o ciberbullying através da educação, deteção e apoio às vítimas.

### 🎯 Motivação

O ciberbullying é uma realidade crescente em Portugal, afetando especialmente crianças e jovens. Este projeto nasceu do desafio lançado pela PSP para criar uma solução digital que ajude a identificar, prevenir e responder a situações de abuso online.

### ✨ Funcionalidades Principais

- 🤖 **Chatbot de apoio** — assistente conversacional powered by IA (Gemini API) que orienta as vítimas, responde a dúvidas e oferece recursos de ajuda
- 📚 **Área educativa** — conteúdos interativos para aprender a reconhecer e prevenir o ciberbullying
- 🔍 **Identificação de situações** — ferramentas para ajudar utilizadores a perceber se estão a ser vítimas de abuso online
- 🆘 **Apoio às vítimas** — recursos, linhas de apoio e orientação sobre como agir em caso de ciberbullying

### 🛠️ Stack Tecnológica

| Camada | Tecnologias |
|---|---|
| Frontend | React + Vite + Tailwind CSS |
| Backend | NestJS + Prisma ORM |
| Base de dados | PostgreSQL |
| IA | Google Gemini API |

### ⚡ Contexto

Projeto desenvolvido em **72 horas** durante um hackathon de março de 2026, em resposta a um desafio real proposto pela PSP. O foco foi criar uma solução funcional, acessível e com impacto social direto.

## 🇬🇧 CiberSeguro

**CiberSeguro** is a web platform developed in 72 hours as part of a hackathon challenge proposed by the **PSP (Portuguese Public Security Police)**, aimed at combating cyberbullying through education, detection, and victim support.

### 🎯 Motivation

Cyberbullying is a growing reality in Portugal, particularly affecting children and young people. This project was born from a challenge issued by the PSP to build a digital solution that helps identify, prevent, and respond to online abuse.

### ✨ Key Features

- 🤖 **Support Chatbot** — AI-powered conversational assistant (Gemini API) that guides victims, answers questions, and provides help resources
- 📚 **Educational Area** — interactive content to help users recognize and prevent cyberbullying
- 🔍 **Situation Identification** — tools to help users understand whether they are experiencing online abuse
- 🆘 **Victim Support** — resources, helplines, and guidance on how to act in cyberbullying situations

### 🛠️ Tech Stack

| Layer | Technologies |
|---|---|
| Frontend | React + Vite + Tailwind CSS |
| Backend | NestJS + Prisma ORM |
| Database | PostgreSQL |
| AI | Google Gemini API |

### ⚡ Context

Project built in **72 hours** during a March 2026 hackathon, in response to a real-world challenge proposed by the PSP. The focus was on delivering a functional, accessible solution with direct social impact.
